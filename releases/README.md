# WeatherMap Android (debug)

- **`WeatherMap-debug.apk`** — Capacitor **debug** build; loads the web app from the URL in `skyline-au/frontend/capacitor.config.ts`. **Home screen / splash icon** matches the PWA artwork (`public/icon-512.png`), regenerated via `npm run android:icons` before builds.

**Sideload:** transfer the APK to your phone → allow installs from that source (e.g. Files / Downloads) → open → Install.

Rebuild: see `skyline-au/README.md` → *Android APK (Capacitor)*.
